Principais formatações

Negrito / destaque

Primeiro colocaremos um termo em negrito utilizando a tag B (não semântica)

Agora, colocaremos um termo em destaque utilizando a tag STRONG (semântico)

Itálico / Ênfase

Nesta frase, iremos colocar um termo em itálico usando a tag I (não semântico)

Enquanto nesta, nós teremos um termo em ênfase usando a tag EM (semântico)

Texto marcado

Podemos também deixar um texto marcado utilizando a tag MARK

Agora temos 2 parágrafos marcado utilizando a mesma tag

Textos grande / pequenos

Podemos deixar qualquer texto um pouco maior que o normal usando a tag BIG (obsoleta)

Como também podemos deixar qualquer texto menor que o normal usando a tag SMALL

Texto deletado / inserido

Nós podemos marcar uma frase como deletada para marcar ela como desconsiderada

E também podemos marcar um texto como sublinhado para dar ênfase nele

Texto sobrescrito

Usamos da tag SUP para escrever coisas do tipo x20 + 4 * 712

Texto subscrito

Podemos usar a tag SUB para escrever coisas do tipo H2SO4
